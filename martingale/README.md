# Martingale Simulation

## Overview
This project evaluates the Martingale betting strategy via Monte Carlo simulation.
It compares outcomes across:
- infinite-bankroll assumptions
- finite-bankroll constraints

The code produces mean and median trajectories with variability bands over many episodes.

## Key Files
- `martingale.py`: simulator and experiment generation
- `p1_results.txt`: experiment summary
- `figure1.png` to `figure5.png`: generated analysis figures

## Skills Demonstrated
- Stochastic process simulation
- Risk-of-ruin analysis
- Experimental design with repeated trials and summary statistics

## Run
```bash
python martingale.py
```
